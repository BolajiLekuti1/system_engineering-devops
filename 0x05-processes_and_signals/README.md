# 0x05. Processes and signals

This directory contains Bash scripts for learning about processes and signals in Linux.

## Requirements
- All files must start with `#!/usr/bin/env bash`
- The second line should be a comment explaining what the script does
- All scripts must be executable

## Files
- `0-what-is-my-pid` - Displays the PID of the script
- `1-list_your_processes` - Lists all currently running processes
